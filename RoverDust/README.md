# KerbalFX - Rover Dust (0.3)

Lightweight rover dust visual module for Kerbal Space Program.

## Depends on

- `KerbalFX Core`

## Config-Driven Tuning

- Global/core values: `GameData/KerbalFX/Core/KerbalFX_Core.cfg` (`KERBALFX_CORE`)
- Module values: `GameData/KerbalFX/RoverDust/KerbalFX_RoverDust.cfg` (`KERBALFX_ROVER_DUST`)

This allows balancing dust density, wheel-size scaling, and light response without rebuilding the DLL.

## Notes

- Designed as an FX-only module (no wheel physics replacement).
- Compatible approach with Parallax/EVE ecosystems.
