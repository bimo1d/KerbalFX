# KerbalFX - Rover Dust

Lightweight wheel-dust visual module for Kerbal Space Program.

## Depends on

- `KerbalFX Core`

## Config-driven tuning

- Shared/core values: `GameData/KerbalFX/Core/KerbalFX_Core.cfg` (`KERBALFX_CORE`)
- Module values: `GameData/KerbalFX/RoverDust/KerbalFX_RoverDust.cfg` (`KERBALFX_ROVER_DUST`)

## Notes

- Designed as an FX-only module (no wheel physics replacement).
- Compatible approach with Parallax.
