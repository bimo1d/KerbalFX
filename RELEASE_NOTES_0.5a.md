## 0.5a

This release introduces the first public version of **KerbalFX - Impact Puffs** and includes major tuning for launch/landing plume behavior. (Very WIP)

### Added
- New module: **Impact Puffs**.
- Engine plume-ground dust puffs for launch and landing.
- One-shot touchdown burst and tuned splashdown-specific behavior.
- Modular slot registration in core config for current/future KerbalFX modules.

### Compatibility
- Built for KSP `1.12.5`.
- Tested alongside Waterfall and Parallax.
